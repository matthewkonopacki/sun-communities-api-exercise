# Sun Communities Example

This is a directory containing both a simple ASP.NET server along with a xUnit test suite for the business logic provided in the server.

## Description

There are two projects contained within this directory. One of them is a simple ASP.NET server that consists of two endpoints detailed in the description of the coding exercise provided. The other is a collection of tests that verifies the fee calculation business logic contained within the server.


## How to Run

For the server:
```bash
dotnet run --project SunCommunitiesApi
```

For the unit tests:
```bash
dotnet test
```
